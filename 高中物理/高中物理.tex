\textbf{电容器描述：}
电容器存储电能的元件，由两个导体之间隔着绝缘材料构成。平行板电容器是由两块平行放置的导体板组成，中间夹有绝缘介质。常用于研究电容器的基本性质。

\textbf{电容器充放电：}
电容器的充电过程是指在外加电压作用下，电容器两极板之间积累电荷的过程；放电过程是指电容器释放储存的电能，电荷流向外部电路的过程。只有在电容器两极板间存在电势差（即外加电源或电压）时，电容器才能充电。充电过程中，电流方向为正极板流入、负极板流出，直到两极板间的电势差等于外加电压时，充电过程结束，电流为零。

\textbf{充电过程中电流和电压的变化：}
电容器刚开始充电时，极板间电压为零，电流最大；随着充电进行，极板间电压逐渐升高，电流逐渐减小；当极板间电压等于外加电压时，电流减小到零，充电过程结束。

电容器存在击穿电压，当电压过大时电容器中间的绝缘材料就会被击穿，使得电容器被破坏。

\textbf{电容器注意事项：}
电容器可以进行串并联，并联电容器，等效电容为电容器电容之和，串联电容器，等效电容规律和并联电阻相同。

\textbf{静电场注意事项：}
\begin{itemize}
  \item $\lambda$为线电荷密度，$\sigma$为面电荷密度，$R$为球体半径，$L$为轴线上距离球心的距离
  \item 电场力做功只与电荷初末位置有关，与路径无关
  \item 电势能增加，一般情况下，机械能减少
  \item 电场力做功时需要进行投影：$W=qEd\cos\theta$
  \item 电场线与等势面垂直
  \item 在匀强电场中，两点所在直线的场强均匀，可以通过$E'=\frac{U}{d}$求出
  \item 固定水平位移时，合速度最小时，水平与竖直方向速度相等
\end{itemize}

\textbf{游标卡尺读数方法：}\\
有两种方法读游标卡尺读数，\\
\textbf{方法一}：下方副尺0刻度线所在位置主尺的读数记为$N$（上图应该是\SI{3}{cm}处，但是不到\SI{31}{mm}）\\
然后找到副尺上与主尺刻度线对齐的刻度线的小格数，记为$m$，\\
游标卡尺读数$=N+m\times \text{游标卡尺精度}$mm\\
如上，为$ 30+6\times 0.1 \text{mm} = 30.6\text{mm}$\\
\textbf{方法二}：找到主尺与副尺刻度线\textbf{对齐}所在位置的读数，记为$N$，\\
找到副尺上与主尺刻度线\textbf{对齐}的刻度线对应的小格数，记为$m$，\\
游标卡尺读数为$=N-\frac{\text{副尺总长度}}{\text{副尺总格子数}}\times m$mm\\
如上，为$ 36 - \frac{6}{10}\times 9 \text{mm} = 30.6\text{mm}$\\
\begin{center}
  \includegraphics[width=0.6\columnwidth]{../assets/螺旋测微器读数.jpg}
\end{center}
\textbf{螺旋测微器读数方法：}\\
先确定右侧旋转筒左侧线所指的主尺半格数，注意上面也有一排刻度线，当超过主尺刻度线时，主尺读数需要加\SI{0.5}{mm}，\\
然后读取右侧旋转筒上与主尺左侧线对齐的刻度线数，乘以螺旋测微器的精度\SI{0.01}{mm}，注意需要估读一位，尽量估读准确一点。\\
最后螺旋测微器读数$=$主尺读数$+$旋转筒读数。\\
如上，为$ 6.5 \text{mm} + 20.3\times 0.01 \text{mm} = 6.703\text{mm}$

\textbf{电路中的基本认识：}
\begin{itemize}
  \item 恒压源的正极电势恒定比负极大$E$
  \item 恒流源干路电流恒定为$I$
  \item 沿电流方向电势降低
  \item 电势降低量$U=RI$
  \item 对电路中任意节点：$\sum I_{in}=\sum I_{out}$
  \item 导线的内阻为0，所以导线可以任意长度变化
  \item 理想电流表视为导线，理想电压表视为断路
  \item 电容器充满电或稳定状态都视为断路
  \item 电容器充电过程视为电阻
  \item 电容器放电视为理想电源
  \item 断路应该被视为是一个无穷大的电阻接在断路的导线两端
  \item 任何部分电阻增大都会导致总电阻增大，反之亦然
\end{itemize}

  \item $\frac{E}{r}$为短路电流
  \item 蓝色直线为电源总功率
  \item 绿色抛物线为电源热功率
  \item 红色抛物线为电源输出功率
  \item 电阻线经过了放大，这样能够清晰地看出输出功率随电阻变化先增大后减小，在内阻等于外阻的时候取得极值
\end{itemize}

\textbf{电功率注意事项：}
\begin{itemize}
  \item 正常工作意味着用电器的全部物理量为额定值
  \item 电动机这样的非纯电阻元器件不能正常工作时，可看作纯电阻
  \item 灯泡虽然会发热，但仍然看成定值电阻
  \item \textbf{不要使用非纯电阻两端电压得到电流或电阻。}$r\neq \frac{U}{I}$
  \item 非纯电阻的内阻也不要与其他电路电阻进行串并联
  \item 通过输出功率最大时电阻情况，此时内外电压相等，大小为$\frac{E}{2}$
  \item 外电阻增大，电源总功率和电源热功率都会减小，电源效率也会升高
  \item 外电压增大，输出功率先增大后减小，当外电压等于$\frac{E}{2}$时，输出功率最大
  \item 此时电源效率为50\%
\end{itemize}

\textbf{电路实验：电表接法选择}

\textbf{电流表的内外接判断：}
\begin{itemize}
  \item 电流表内接时，因为电流表分压，电阻测量值偏大
  \item 电流表外接时，因为电压表分流，电阻测量值偏小
\end{itemize}

\textbf{判断电表接法方法一：临界值法}
\begin{itemize}
  \item 如果$R_x<\sqrt{R_VR_A}$，采用外接法
  \item 如果$R_x>\sqrt{R_VR_A}$，采用内接法
\end{itemize}

\textbf{判断电表接法方法二：试触法}
\begin{itemize}
  \item 如果两次内外接过程中，电流表读数变化较大，即电压表分流明显，采用内接法
  \item 如果电压表读数变化较大，即电流表分压明显，采用外接法
\end{itemize}

\textbf{判断电表接法方法三：内阻已知法}
\begin{itemize}
  \item 电流表内阻已知，采用内接法
  \item 电流表内阻未知，采用外接法
\end{itemize}

\textbf{电表量程选择：}
\begin{itemize}
  \item 可以略超量程，但是不能选择过大的量程
  \item 例如电源电压为4V，选择3V的电压表是可行的
  \item 但是选择15V的电压表就不合适了
\end{itemize}

\textbf{电路实验：滑动变阻器选取原则}

按照如下原则依次进行：
\begin{itemize}
  \item 要求大范围，电压或电流从0开始，测量用电器伏安特性曲线，分压式接法
  \item 滑动变阻器总电阻小于用电器，分压式接法
  \item 滑动变阻器总电阻和用电器接近，限流式接法
  \item 需要节省能量，尽可能节约，限流式接法
\end{itemize}

\textbf{滑动变阻器阻值选择简化版：}
\begin{itemize}
  \item 限流式：$R_{\text{滑动变阻器}}$略大于或约为 $3\sim 5$ 倍 $R_x$
  \item 分压式：$R_{\text{滑动变阻器}}<R_x$（需要电压能从0开始）
\end{itemize}

\textbf{半偏法测电阻一定有误差}

\textbf{限流式电路中的半偏法：}
\begin{itemize}

  \item 但先满偏后半偏时，干路电流将会变大

  \item 两个已知阻值的定值电阻（或者使用均匀的电阻丝，通过长度比值得到电阻之比）
  \item 一个电阻箱
\end{itemize}

\textbf{磁场力注意事项：}
\begin{itemize}
  \item 地磁场水平方向上从南指向北
  \item 竖直方向北半球有向下的分量，南半球有向上的分量
  \item 磁场是真实存在的特殊物质，磁感线是人为创造的模型
  \item 磁感线在磁体外部从N极指向S极，在磁体内部从S极指向N极
  \item 小磁针N极受力方向为磁场方向
  \item 安培定则：右手握住导线，大拇指指向电流方向，此时四指环绕方向为磁场方向
  \item 右手螺旋定则：右手握住螺线管，四指环绕方向为电流方向，大拇指指向磁体内部磁场方向
  \item 同向电流导线相互吸引，反向电流导线相互排斥
  \item 左手定则：四指伸直指向电流方向，磁感线穿过掌心，大拇指指向安培力方向
  \item 安培力方向与电流方向和磁场方向都垂直，电流方向和磁场方向垂直时安培力最大
  \item 左手定则中四指指向正电荷移动方向与负电荷移动反方向
  \item 洛伦兹力方向与速度方向和磁场方向都垂直，速度方向和磁场方向垂直时洛伦兹力最大
  \item 洛伦兹力不能影响电荷的速度大小，只能改变电荷的运动方向
  \item 洛伦兹力可以通过影响正压力的方式影响滑动摩擦力，进而影响速度
  \item 沿电流方向速度引起的洛伦兹力分力是安培力的微观本质
  \item 洛伦兹力分力可以做功，影响物体的能量分配
\end{itemize}

\textbf{电磁应用注意事项：}
\begin{itemize}
  \item 速度选择器提高精度可以增加板长、减小板间距
  \item 速度选择器只能从一个方向进入
  \item 速度选择器无法分辨电性
  \item 质谱仪根据不同的比荷得到不同的半径
  \item 质谱仪中比荷越大半径越小；比荷越小半径越大
  \item 霍尔电压中，载流子的电性直接影响电压的正负极
  \item 回旋加速器中，电场加速时间忽略不计
  \item 回旋加速器加速总时间和粒子的比荷无关
  \item 回旋加速器加速不同比荷的粒子需要调整电压频率
\end{itemize}

\textbf{带电粒子运动注意事项：}
\begin{itemize}
  \item 直线性边界下，进入夹角等于离开夹角，速度偏转角为$2\theta$

\textbf{楞次定律}

感应电流（或感应电动势）的方向总是阻碍引起感应电流的磁通量变化。判断方向时可先判断原磁通量如何变化，再确定“阻碍变化”的磁场方向，最后用右手定则判断感应电流方向。
楞次定律最关键的不是“阻碍磁场”，而是“阻碍磁通量的变化”。为了方便做题，常总结成几个非常常用的口诀。

\begin{itemize}
  \item 来拒去留：磁体靠近线圈时，感应电流的磁场阻碍靠近；磁体远离线圈时，感应电流的磁场阻碍远离，所以表现为“来时排斥，去时吸引”
  \item 增缩减扩：原磁通量增大时，感应电流的磁场要阻碍增大，所以回路有缩小趋势；原磁通量减小时，感应电流的磁场要阻碍减小，所以回路有扩张趋势
  \item 增反减同：原磁场增强时，感应电流产生的磁场方向与原磁场相反；原磁场减弱时，感应电流产生的磁场方向与原磁场相同
\end{itemize}

上面这些口诀本质上说的都是同一件事，也就是感应电流总要让磁通量“尽量维持原状”。所以做题时不要死记口诀，而是先判断穿过回路的磁通量是增大还是减小，再判断感应电流应该产生什么方向的磁场，最后再确定电流方向。

\textbf{电磁感应定律}

法拉第定律定量给出感应电动势大小，楞次定律定性给出方向。两者要结合起来用：先用法拉第定律求大小，再用楞次定律判断方向。对一般情况有
\[
  E=n\left|\frac{\Delta \Phi}{\Delta t}\right|.
\]
其中引起磁通量变化的原因可以不同，因此感应电动势又常分为动生电动势和感生电动势。

\textbf{动生与感生}

动生电动势，指的是导体或回路的某一部分在磁场中运动，由“切割磁感线”产生的感应电动势。本质上是导体中的自由电荷随导体运动，受到洛伦兹力而定向移动形成的。最常见的公式就是导体棒切割磁感线：
\[
  E=BLv.
\]

感生电动势，指的是回路本身并不一定运动，而是穿过回路的磁场随时间变化，导致磁通量变化，从而产生感应电动势。这时更适合直接使用
\[
  E=n\left|\frac{\Delta \Phi}{\Delta t}\right|.
\]
典型特征是“磁场在变，线圈不一定动”。

需要注意的是，动生和感生只是产生感应电动势的两种常见形式，它们都服从法拉第电磁感应定律。也就是说，不管是导体在动，还是磁场在变，最终都可以归结为“磁通量发生变化，因而产生感应电动势”。

\textbf{动生与感生的联系}

\begin{itemize}
  \item 只要磁通量变化，就可能产生感应电动势
  \item 磁通量变化可以由回路运动引起，也可以由磁场变化引起
  \item 动生和感生都满足法拉第电磁感应定律，方向都由楞次定律判断
  \item 有些综合题中，动生和感生可以同时存在，此时要看总磁通量如何变化
\end{itemize}

\textbf{常见例子}

\begin{itemize}
  \item 导体棒在平行导轨上匀速切割磁感线，产生感应电动势$E=BLv$，这是典型的动生电动势
  \item 闭合线圈静止在变化的匀强磁场中，磁感应强度随时间增大或减小，线圈中产生感应电流，这是典型的感生电动势
  \item 条形磁铁插入或拔出线圈时，线圈本身可以不动，但由于穿过线圈的磁通量变化，线圈中产生感应电流，这也属于感生电动势
  \item 矩形线圈在匀强磁场中绕轴转动，线圈边切割磁感线，磁通量也在周期变化，这类题通常按转动切割磁感线处理，属于动生电动势的典型模型
\end{itemize}

所以做题时，可以按下面顺序判断：先看回路中的哪一部分在变，是“导体在动”还是“磁场在变”；再判断磁通量是否变化；接着用法拉第定律求感应电动势大小；最后用楞次定律判断方向。如果题目能直接看出单根导体棒切割磁感线，优先用$E=BLv$；如果是整个回路的磁通量随时间变化，优先用$E=n\left|\frac{\Delta\Phi}{\Delta t}\right|$。

\textbf{电动机模型}

电动机模型通常与安培力和能量转化结合：导体在磁场中受力转动，电能转化为机械能并伴随焦耳热。分析时可结合$P=UI$、$P_{\text{热}}=I^2R$与输出功率关系。
如果研究的是线圈在匀强磁场中绕固定轴转动，并要求感应电动势，那么本质上就是转动切割磁感线问题。此时既要会判断磁通量变化，也要会判断线圈中哪一边切割更强，从而确定电动势的大小和正负。

\textbf{中性面}

中性面是这类题的核心概念。所谓中性面，就是线圈平面与磁感线垂直的位置，也就是线圈法线方向与磁场方向平行的位置。在线圈经过中性面时，穿过线圈的磁通量最大，虽然此时磁通量数值最大，但磁通量变化率恰好为0，所以感应电动势为0。

与中性面对应的另一个特殊位置，是线圈平面与磁感线平行的位置。此时穿过线圈的磁通量为0，但磁通量变化最快，所以感应电动势最大。

因此可以总结出两个最重要的结论：
\begin{itemize}
  \item 过中性面时，感应电动势为0
  \item 线圈平面与磁感线平行时，感应电动势最大
\end{itemize}

\textbf{感应电动势大小}

设线圈面积为$S$，匝数为$n$，在线圈法线与磁场方向夹角为$\theta$时，磁通量为
\[
  \Phi=BS\cos\theta.
\]
若线圈以角速度$\omega$匀速转动，则$\theta=\omega t+\theta_0$，由法拉第电磁感应定律可得
\[
  e=nBS\omega\sin\theta.
\]
若从中性面开始计时，则$\theta=\omega t$，此时常写成
\[
  e=nBS\omega\sin\omega t.
\]
其中感应电动势最大值为
\[
  E_m=nBS\omega.
\]
所以转动过程中感应电动势并不是恒定的，而是按正弦规律周期变化。经过中性面时电动势为0，转过四分之一周期到线圈平面与磁场平行时，电动势达到最大；再转过四分之一周期回到另一侧中性面时，电动势又变为0。

\textbf{感应电动势正负}

感应电动势的正负本质上反映的是电流方向或者线圈两端电势高低的变化。要判断正负，必须先规定正方向，例如先规定某一端为电动势正方向，或者规定线圈中某一环流方向为正方向。规定以后，再根据楞次定律或右手定则判断实际方向，和规定方向相同取正，相反取负。

在线圈连续转动一周的过程中，感应电动势方向会周期性改变。每经过一次中性面，线圈切割磁感线的方向就整体反向一次，因此感应电动势也会变号一次。也就是说：
\begin{itemize}
  \item 一个周期内，感应电动势有两次为0，都是在线圈经过中性面时
  \item 一个周期内，感应电动势有一次最大正值和一次最大负值
  \item 线圈每经过一次中性面，感应电动势方向改变一次
\end{itemize}

如果从中性面开始计时，并规定刚离开中性面时的感应电动势方向为正，那么前半周内电动势先增大到正最大值，再减小到0；后半周内电动势再增大到负最大值，最后回到0。所以交流发电机输出的电动势本质上就是这样产生的。

做题时判断这类问题的顺序一般固定：先找中性面，判断特殊位置处电动势是0还是最大；再由$E_m=nBS\omega$求最大值；最后结合楞次定律或右手定则判断某一时刻电动势的正负。

该模型一般有初速度或者外力，外力研究一般让其不变，如果是可变外力，那么很有可能是与安培力相等的，或者让单棒能够匀变速运动的外力。结合题目调整外力的表达式即可，研究我们以恒定外力为例。
接着以同时具有初速度与恒力的单棒为例进行分析，其中初速度的方向与外力的方向是指向电阻还是背对电阻不影响，（固定电阻的情况下）。那么问题可以首先分成两类，初速度方向与外力方向一致与不一致。
这两种情况有一点是相同的，安培力一定时刻与速度方向相反，这很容易通过楞次定律判断。接着第一种一致的情况，需要判断安培力的大小与外力的大小关系，如果安培力大于外力，那么单棒做减速运动，直到安培力等于外力，单棒做匀速运动。
列出单棒的牛顿运动定律、动能定理、动量定理。首先是牛顿运动定律：如果外力大于安培力，$F-F_{\text{安}}=ma$，接着是动能定理：$Fx-F_{\text{安}}x=\frac{1}{2}mv^2-\frac{1}{2}mv_0^2$，最后是动量定理：$Ft-F_{\text{安}}t=mv-mv_0$。其中安培力做功无法直接求出，但是在这种情况下，安培力一定做负功，并且就是回路中产生的焦耳热。在动量定理中，安培力的冲量有两种表达式，也就是$I_{\text{安}}=BLq=\frac{B^2L^2x}{R}$。
如果外力小于安培力，$F_{\text{安}}-F=ma$，接着是动能定理：$F_{\text{安}}x-Fx=\frac{1}{2}mv^2-\frac{1}{2}mv_0^2$，最后是动量定理：$F_{\text{安}}t-Ft=mv-mv_0$。和前面类似注意一下大小关系即可。
如果安培力小于外力，那么单棒做加速运动，直到安培力等于外力，单棒做匀速运动。两者的最终匀速速度可以直接得到，$v=\frac{F}{B^2L^2}R$。在反向之后可以看成初速度为0的，公式和前面的一样，关注一下减速到0这个过程。此时牛顿运动定律是$F+F_{\text{安}}=ma$，动能定理是$F_{\text{安}}x+Fx=\frac{1}{2}mv^2$，动量定理是$F_{\text{安}}t+Ft=mv$。注意一下大小关系即可。

第二种情况，外力与初速度方向相反与安培力方向相同，那么单棒做减速运动，一直到速度为0，然后开始反向加速，直到安培力等于外力，单棒做匀速运动。两者的最终匀速速度可以直接得到，$v=\frac{F}{B^2L^2}R$。
最终你会发现，只要有外力，单棒最终都会做匀速运动，并且匀速速度为$v=\frac{F}{B^2L^2}R$。那反之，如果外力为0，那么单棒最终都会停下来。

电源模型相对复杂得多，原因很简单，电源会施加一个额外的电流，使得不能直接通过运动方向得出安培力与速度反向。
单棒可能有初速度可能没有，可能有外力可能没有。下面研究的时候以全都存在作为例子。
先进行问题的分类讨论，首先是初速度方向产生的感应电动势可能与电源的相同或者相反，外力可能与电源产生的电流对应的安培力方向相同或者相反，这样就是四种情况。我们后面可以发现其实四个过程中存在一个先后发生而非并列的关系，类似于前面电阻模型中的先减速后加速过程。那最初研究我们就不做这样的前瞻性判断，直接分成四种不同的情况去研究。
第一种情况，初速度产生的感应电动势与电源的电动势相叠加，也就是正极与负极相连接。并且外力与电源产生的电流对应的安培力方向相同。此时初速度产生的安培力与初速度方向相反，也就与外力方向相同。此时单棒会减速运动，减速后安培力减小，但是外力仍然存在则并不会匀速，而是一直减小到0，接着外力和电源提供的安培力会使得单棒反向加速，而感应电动势此时因为速度反向而与电源的电动势抵消，单棒的加速度在逐渐减小，等到两个电动势相等时，单棒的安培力等于0，单棒此时只收到外力，继续加速。感应电动势进一步增大，大于电源电动势，安培力也开始反向增大，逐渐增大到与外力相等时，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(BLv-E)}{R}$。进一步求出速度即可。所以你会发现$E=BLv$是一个特殊值，但并不是匀速运动的那个条件，因为外力存在导致的，如果此类问题没有外力，那么匀速运动的条件就是$E=BLv$。
第二种情况，初速度产生的感应电动势与电源的电动势相叠加，也就是正极与负极相连接。并且外力与电源产生的电流对应的安培力方向相反。此时外力一开始就与合成的安培力相反，单棒具体运动状态需要观察一下安培力的大小关系，
如果安培力大于外力，那么单棒做减速运动，而减速的安培力随着速度的减小在减小，
有可能外力直到速度等于0也没有大于等于安培力，也就是条件：$F\leq \frac{BLE}{R}$。这样单棒一定会减速到0，然后反向运动。
还有可能减小并且外力比电源提供的安培力大一点，所以以另一种形式匀速：$F=\frac{BL(BLv+E)}{R}$。
另一种即是外力较大($F>\frac{BL(BLv_0+E)}{R}$)，单棒做的是加速运动，加速直到安培力等于外力，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(BLv+E)}{R}$。进一步求出速度即可。
如果此时外力已经等于安培力了，单棒就直接做匀速运动了，最终速度可以求出：$F=\frac{BL(BLv_0+E)}{R}$。进一步求出速度即可。
第三种情况，初速度产生的感应电动势与电源的电动势相抵消，也就是正极与正极相连接。并且外力与电源产生的电流对应的安培力方向相同。初速度产生的安培力与外力和电源提供的安培力合力方向相反，需要比较大小关系。
如果合力大于感应的安培力，那么单棒做加速运动，感应的安培力增大，直到安培力等于合力，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(BLv-E)}{R}$。进一步求出速度即可。
如果合力小于感应的安培力，那么单棒做减速运动，感应的安培力减小，直到安培力等于合力，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(BLv-E)}{R}$。进一步求出速度即可。
所以会发现，两种情况下匀速的速度固定，并且只与外力和电源的关系决定，到这里实际上我们也可以得到一些固定结论了，所有的电源模型，都可以将外力与电源进行合成，变成一个外力，这是因为电源会提供稳定电流，进而提供稳定的安培力，这样相等于变成了一个纯电阻与恒定外力加初速度的模型。
第四种情况，初速度产生的感应电动势与电源的电动势相抵消，也就是正极与正极相连接。并且外力与电源产生的电流对应的安培力方向相反。按照上面的理论，应该是电源安培力减去外力的合力与感应安培力进行比较，如果合力大于感应安培力，那么单棒做加速运动，感应的安培力增大，直到安培力等于合力，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(E-BLv)}{R}$。进一步求出速度即可。
如果合力小于感应安培力，那么单棒做减速运动，感应的安培力减小，直到安培力等于合力，单棒做匀速运动。最终速度可以求出：$F=\frac{BL(E-BLv)}{R}$。进一步求出速度即可。

电容器模型的电流不好表示，其他两个模型可以通过欧姆定律求出电流。电容器需要使用电流的定义式，并且电容器可以方便地求出充放电的电流大小。电流等于电荷量的变化率，$I=\frac{\Delta Q}{\Delta t}=\frac{C\Delta U}{\Delta t}=C\frac{\Delta(BLv)}{\Delta t}=CBL\frac{\Delta v}{\Delta t}=CBLa$，也就是电流大小与加速度成正比，再计算单棒受到的安培力，$F_{\text{安}}=BLI=CB^2L^2a$，可以看出安培力与加速度成正比，单棒的运动方程为$ma=F-CB^2L^2a$，可以求出加速度为$a=\frac{F}{m+CB^2L^2}$，可以看出电容器的存在相当于增加了单棒的质量，所以单棒的加速度会变小。我们是如何确定电容器的安培力方向的呢？可以把充电状态的电容器看成一个纯电阻，这样就可以通过电阻模型来判断安培力的方向了。
而如果电容器带有初始电荷量，此时电容器可以通过$U=\frac{Q}{C}$来表示电压，再把电容器看成一个变化电动势的电源。其余的分析可以参照电源模型，此处不赘述了。
\textbf{双棒模型}

双棒模型大致可以分成两大类，安培力相等和安培力不等。
这两种情况最主要的区别在于安培力的冲量是否相同，所以我并没有根据导轨来判断。就算导轨相同，也可以通过磁场强度使得安培力不相等，此时和导轨不同是一样的作用了。BL总是在一起相乘，所以我才会按照安培力进行区分。
第一种情况，安培力冲量相等，两个单棒类似于直接通过绳子相连，只不过这里的“绳子”不是机械绳，而是电磁感应带来的相互制约。若两棒始终处在同一回路中，并且$BL$乘积相同，那么任意时刻两棒受到的安培力大小相等、方向相反。对于整体来说，安培力属于内力，分析整体运动时可以消去；对于单棒来说，安培力又真实存在，会改变各自的速度，所以这种模型最适合同时研究“单棒”和“整体”。

先说几个稳定结论。第一，安培力总是阻碍两棒间的相对运动，也就是阻碍回路面积变化。第二，若系统不受外力，则整体动量守恒，最终一定趋向于共速运动。第三，从能量角度看，两棒相对运动的机械能会不断转化为回路中的焦耳热，所以双棒模型本质上是“动量守恒和能量损失同时存在”的模型。

下面把安培力相等时分成几种常见情形。

\textbf{第一类：两根单棒初速度相同}

若两棒初速度大小和方向都相同，那么它们之间没有相对运动，回路面积不变，感应电动势为0，电流为0，因此安培力也为0。此时两棒实际上就像两个彼此没有相互作用的物体。

从受力看，每根棒都不受安培力；若题目没有额外外力，则每根棒都只受重力和支持力，在导轨方向上合力为0。

从运动看，两棒始终保持相同速度做匀速运动，整体也做匀速运动，不会再发生速度重新分配。

从动量定理看，对任意一根棒都有合外力冲量为0，所以$m_1v_1-m_1v_{10}=0$，$m_2v_2-m_2v_{20}=0$；对整体也有总冲量为0，所以
\[
  m_1v_1+m_2v_2=m_1v_{10}+m_2v_{20}.
\]

从动能定理看，由于没有安培力做功，也没有外力在导轨方向做功，因此两根棒和整体的动能都不变，即
\[
  \Delta E_{k1}=0,\quad \Delta E_{k2}=0,\quad \Delta E_{k\text{总}}=0.
\]

\textbf{第二类：两根单棒初速度相反}

若两棒初速度方向相反，或者虽然同向但大小不同，那么本质上都是存在相对运动。只要有相对运动，回路磁通量就会变化，回路中就会产生感应电流，两棒受到大小相等、方向相反的安培力。安培力的作用效果永远是减小相对速度，所以一个棒被拉慢，另一个棒被带快，最后两棒达到共同速度。

从受力看，设开始时棒1沿正方向运动更“占优势”，则棒1受到的安培力方向与其速度方向相反，棒2受到的安培力方向与其最终趋向的共同运动方向一致。若不计其他外力，则导轨方向上棒1只受安培力，棒2也只受安培力，且二者大小始终相等。

从运动看，系统会经历“相对速度减小”的过程。若两棒初速度反向，通常表现为一根棒减速，另一根棒可能先减速到0再反向，最终两棒同向共速。最后的共同速度由整体动量决定：
\[
  v=\frac{m_1v_{10}+m_2v_{20}}{m_1+m_2}.
\]
这里$v_{20}$若取负值，就已经自动表示“初速度相反”。

从单棒的动量定理看，安培力冲量大小相等。设安培力冲量大小为$I_{\text{安}}$，则
\[
  -I_{\text{安}}=m_1v-m_1v_{10},
\]
\[
  I_{\text{安}}=m_2v-m_2v_{20}.
\]
也就是说，两棒动量变化量等大反向。

从整体的动量定理看，由于安培力是内力，整体在导轨方向不受外力时总动量守恒，即
\[
  m_1v_1+m_2v_2=m_1v_{10}+m_2v_{20}.
\]
在最终共速时就得到上面的共同速度公式。

从单棒的动能定理看，棒1上安培力一般做负功，棒2上安培力可能做正功，但两棒安培力做功代数和一定为负。若分别记安培力对两棒做功为$W_{\text{安}1}$、$W_{\text{安}2}$，则
\[
  W_{\text{安}1}=\Delta E_{k1},\quad W_{\text{安}2}=\Delta E_{k2}.
\]
对整体使用动能定理更清楚：
\[
  W_{\text{安总}}=\Delta E_{k\text{总}}.
\]
而整体安培力总功不为0，它等于回路中损失的电能的相反数，所以
\[
  \Delta E_{k\text{总}}=-Q.
\]
把初末状态代入，可得
\[
  \frac{1}{2}m_1v_{10}^2+\frac{1}{2}m_2v_{20}^2-\frac{1}{2}(m_1+m_2)v^2=Q.
\]
这说明减少的机械能全部转化为焦耳热。

\textbf{第三类：其中一根单棒受到外力}

这是最常考的一类。设外力$F$作用在棒1上，方向沿导轨。安培力仍然满足两棒大小相等、方向相反，因此分析时要同时抓住两条线索：对单棒看安培力，对整体看外力。

从受力看，棒1在导轨方向受到外力$F$和安培力$F_{\text{安}}$，棒2只受到安培力$F_{\text{安}}$。安培力方向仍然遵循“阻碍相对运动”。若棒1在外力作用下有比棒2更大的前进趋势，则棒1所受安培力向后，棒2所受安培力向前。

从运动看，外力使整体获得加速度，而安培力负责在两棒之间传递作用，使棒2也被“带动”起来。若外力恒定，且系统电阻等因素不变，那么两棒的相对速度一般先变化后减小，最终常常趋于相对静止，此后两棒以共同加速度或共同速度运动。也就是说，安培力相当于把外力对一根棒的作用逐渐传递给另一根棒。

从单棒的牛顿运动定律看，若取外力方向为正方向，并且此时棒1相对更快，则有
\[
  F-F_{\text{安}}=m_1a_1,
\]
\[
  F_{\text{安}}=m_2a_2.
\]
若某一阶段安培力方向需要反向，只需在式中改写正负号即可，核心仍然是“安培力总是阻碍相对运动”。

从整体的牛顿运动定律看，安培力抵消后只剩外力，所以
\[
  F=(m_1+m_2)a_{\text{整体}}.
\]
这说明整体的加速度只由外力决定，与安培力无关；安培力只影响两棒之间的速度分配。

从单棒的动量定理看，对棒1有
\[
  Ft-I_{\text{安}}=m_1v_1-m_1v_{10},
\]
对棒2有
\[
  I_{\text{安}}=m_2v_2-m_2v_{20}.
\]
两式相加即可得到整体的动量定理：
\[
  Ft=(m_1v_1+m_2v_2)-(m_1v_{10}+m_2v_{20}).
\]
可以看出，整体动量的改变只由外力冲量决定。

从单棒的动能定理看，对棒1有
\[
  Fx_1+W_{\text{安}1}=\frac{1}{2}m_1v_1^2-\frac{1}{2}m_1v_{10}^2,
\]
对棒2有
\[
  W_{\text{安}2}=\frac{1}{2}m_2v_2^2-\frac{1}{2}m_2v_{20}^2.
\]
这里$W_{\text{安}1}$、$W_{\text{安}2}$的正负仍然由各自受力方向与位移方向决定。

从整体的动能定理看，最简洁的写法是
\[
  Fx_1=\Delta E_{k\text{总}}+Q.
\]
也就是说，外力做的功一部分转化为整体机械能的增加，另一部分转化为焦耳热。若最终两棒达到共同速度$v$，则末态总动能为
\[
  E_{k\text{末}}=\frac{1}{2}(m_1+m_2)v^2.
\]
因此这类问题通常用“整体动量定理求速度关系，用整体动能定理或能量守恒求焦耳热”会最简洁。

综上，安培力相等的双棒模型，本质上就是“内力等大反向”的双物体模型。没有外力时，整体动量守恒，最终共速；有外力时，整体只看外力，单棒再单独看安培力。解题时优先画清楚两棒的安培力方向，再分别列单棒和整体的动量、能量关系，通常就不会乱。

第二种情况，安培力不等时，就不能再把它简单看成“绳子模型”了，因为两棒受到的安培力大小可以不相等，安培力冲量也不相等。这样一来，对单棒来说仍然可以分别研究安培力；但对整体来说，两棒安培力的冲量代数和不再为0，所以整体动量一般不再守恒，这就是和前一种情况最本质的区别。

从受力看，两根棒在导轨方向上分别受到各自的安培力，必要时再叠加题目给出的外力。安培力方向仍然遵循楞次定律，始终阻碍回路面积的变化或者阻碍相对运动的趋势，只是由于两棒所在回路位置、磁场强弱、有效长度或者电阻结构不同，两棒受到的安培力大小不再相同。

从运动看，系统不再表现为简单的“相互拉着走”，而更像是两个通过电磁联系互相影响、但影响强弱不同的物体。哪根棒所受安培力更大，哪根棒的速度变化就更明显。最终是否共速、是否匀速，要看题目中是否还有外力以及两棒安培力如何随速度和位移变化，不能直接套用前面安培力相等时的结论。

从单棒的动量定理看，仍然是对两根棒分别列式。若棒1、棒2受到的安培力冲量分别为$I_{\text{安}1}$、$I_{\text{安}2}$，再考虑可能存在的外力冲量$I_{\text{外}1}$、$I_{\text{外}2}$，则有
\[
  I_{\text{外}1}+I_{\text{安}1}=m_1v_1-m_1v_{10},
\]
\[
  I_{\text{外}2}+I_{\text{安}2}=m_2v_2-m_2v_{20}.
\]
这里最关键的是
\[
  I_{\text{安}1}+I_{\text{安}2}\neq 0.
\]

因此对整体列动量定理时，不能把安培力直接消掉，而应写成
\[
  I_{\text{外总}}+I_{\text{安总}}=(m_1v_1+m_2v_2)-(m_1v_{10}+m_2v_{20}),
\]
其中$I_{\text{安总}}=I_{\text{安}1}+I_{\text{安}2}$。所以整体速度变化不仅由外力决定，也会受到安培力总冲量的影响。

从单棒的动能定理看，仍然分别研究每根棒：
\[
  W_{\text{外}1}+W_{\text{安}1}=\frac{1}{2}m_1v_1^2-\frac{1}{2}m_1v_{10}^2,
\]
\[
  W_{\text{外}2}+W_{\text{安}2}=\frac{1}{2}m_2v_2^2-\frac{1}{2}m_2v_{20}^2.
\]
对整体来说，则有
\[
  W_{\text{外总}}+W_{\text{安总}}=\Delta E_{k\text{总}}.
\]
而安培力总功通常仍与回路中的焦耳热有关，做题时一般仍然要结合功能关系或者能量守恒去分析。

所以安培力不等的双棒模型，方法上和前面一致，仍然是“先判断安培力方向，再分别分析单棒和整体”；只是此时对整体不能再默认安培力是内力且冲量为0，必须把安培力总冲量和安培力总功保留下来。

\textbf{微观解释}

体积、压强和温度都可以从分子热运动角度理解：气体分子无规则运动并不断碰撞容器壁，形成压强；温度反映分子热运动的剧烈程度。

\textbf{分子动理论}

物体由大量分子组成，分子在永不停息地做无规则运动，分子间同时存在引力和斥力。热现象的宏观规律由大量分子的统计行为决定。

\subsection{热力学定律}
\begin{tabularx}{\columnwidth}{XX}
  \rowcolor{blue!30}
  \multicolumn{2}{l}{
    \parbox[c][2.5\ht\strutbox][c]{0.9\columnwidth}{\Large 热力学定律}
  }\\
  热力学第一定律&$\Delta U=Q+W$\\
  等容过程&$W=0$\\
  等压过程&$W=p\Delta V$\\
  等温过程&$\Delta U=0$\\

\end{tabularx}

\textbf{第一定律}

第一定律给出内能变化与做功、热传递之间的关系。等容过程中系统不对外做功；等压过程中可用$W=p\Delta V$计算体积功；等温过程中理想气体内能不变；绝热过程中系统与外界无热交换。

\textbf{自由膨胀}

自由膨胀过程中气体失去外界压强约束，通常视为不做功。

\textbf{第二定律}

热量不能自发地从低温物体传向高温物体；不可能从单一热源吸收热量并全部转化为有用功。

\textbf{第三定律}

绝对零度无法达到。

\textbf{第零定律}

当系统之间停止热交换时，它们达到热平衡，温度相同。
