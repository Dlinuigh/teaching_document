
\subsection{光电效应}
\begin{tabularx}{\columnwidth}{XX}
  爱因斯坦光电方程&$h\nu=W_0+E_{k\max}$\\
  最大初动能（电压表示）&$E_{k\max}=eU_c$\\
  截止频率&\(\nu_0=\frac{W_0}{h}\)\\
  逸出功（波长表示）&\(W_0=\frac{hc}{\lambda_0}\)\\
  最大初动能--频率图像&\(E_{k\max}=h\nu-W_0=h(\nu-\nu_0)\)\\
  遏止电压--频率图像&\(U_c=\frac{h}{e}\nu-\frac{W_0}{e}=\frac{h}{e}(\nu-\nu_0)\)\\
\end{tabularx}

\begin{formulanotebox}
是否发生光电效应由频率是否达到截止频率决定；光强主要影响单位时间逸出电子数，不决定单个电子最大初动能。

截止电压也常称遏止电压（或遏制电压），$U_c$ 对应“恰好遏止最大初动能电子”，不是所有光电子都恰好具有该动能。
\end{formulanotebox}

\begin{keypointbox}
\begin{itemize}[leftmargin=1.5em]
  \item 光电效应结论体现“光的粒子性”：是否发生光电效应由入射光频率是否超过截止频率决定，而不是由光强单独决定。
  \item 光电子最大初动能满足 $E_{k\max}=h\nu-W_0$，频率越高，单个光电子可获得的最大初动能越大。
  \item 遏止电压满足 $eU_c=E_{k\max}$，因此 $U_c$ 只随入射光频率和金属材料改变，与光强无关。
\end{itemize}
\end{keypointbox}

\begin{casetypebox}[title={\strut\textcolor{white}{\bfseries 高频结论：光电效应实验现象}}]
\begin{itemize}[leftmargin=1.5em]
  \item 存在截止频率：$\nu<\nu_0$ 时无光电子逸出。
  \item 存在饱和电流：增大光强会增大饱和电流。
  \item 光电子最大初动能与光强无关，仅由入射频率和逸出功决定。
  \item 增大入射频率会增大最大初动能，也会提高截止电压。
  \item 光电效应几乎瞬时发生，符合光子与电子单次作用模型。
\end{itemize}
\end{casetypebox}

\begin{casetypebox}[title={\strut\textcolor{white}{\bfseries 常见题型：光电效应图像分析}}]
\textbf{$E_{k\max}$--$\nu$ 图像：}
\begin{itemize}[leftmargin=1.5em]
  \item 图线方程为 $E_{k\max}=h\nu-W_0$，斜率为普朗克常量 $h$。
  \item 横轴截距为截止频率 $\nu_0$，满足 $W_0=h\nu_0$。
  \item 纵轴截距为 $-W_0$；同一金属换强光时，图线不改变，只改变光电流大小。
\end{itemize}

\textbf{$U_c$--$\nu$ 图像：}
\begin{itemize}[leftmargin=1.5em]
  \item 图线方程为 $U_c=\frac{h}{e}\nu-\frac{W_0}{e}$，斜率为 $\frac{h}{e}$。
  \item 横轴截距仍为截止频率 $\nu_0$；纵轴截距为 $-\frac{W_0}{e}$。
  \item 不同金属的斜率相同，截止频率和纵轴截距不同；由图像可求 $h$、$W_0$、$\nu_0$。
\end{itemize}
\end{casetypebox}
