# 高中数学：数列

## 一、数列基础概念

### 1.1 数列的定义
数列是按照一定规律排列的一列数。通常用 $\{a_n\}$ 表示数列，其中 $a_n$ 表示第 $n$ 项（通项）。

**分类：**
- **有穷数列**：项数有限
- **无穷数列**：项数无限
- **递增数列**：$a_n < a_{n+1}$
- **递减数列**：$a_n > a_{n+1}$
- **常数列**：所有项相等

### 1.2 通项公式与递推关系
- **通项公式**：$a_n = f(n)$，直接用 $n$ 表示第 $n$ 项
- **递推关系**：通过前一项或前几项表示后一项，如 $a_{n+1} = a_n + d$

## 二、等差数列 (Arithmetic Progression)

### 2.1 定义
如果一个数列从第二项起，每一项与前一项的差都等于同一个常数 $d$，则称该数列为**等差数列**，$d$ 称为**公差**。

$$a_{n+1} - a_n = d \quad (n \in \mathbb{N}^*)$$

### 2.2 通项公式

设等差数列首项为 $a_1$，公差为 $d$，则第 $n$ 项为：

$$a_n = a_1 + (n-1)d$$

**推导**：
- $a_1 = a_1$
- $a_2 = a_1 + d$
- $a_3 = a_1 + 2d$
- $\vdots$
- $a_n = a_1 + (n-1)d$

### 2.3 性质

**性质1**：若 $m + n = p + q$，则 $a_m + a_n = a_p + a_q$

**推导**：
$$a_m + a_n = [a_1 + (m-1)d] + [a_1 + (n-1)d] = 2a_1 + (m+n-2)d$$
$$a_p + a_q = [a_1 + (p-1)d] + [a_1 + (q-1)d] = 2a_1 + (p+q-2)d$$

当 $m+n = p+q$ 时，两式相等。

**性质2**：三个数成等差数列 $\Leftrightarrow$ 中间一个是另两个的等差中项

即若 $a, b, c$ 成等差数列，则 $b = \frac{a+c}{2}$

### 2.4 前 $n$ 项和公式

设等差数列的前 $n$ 项和为 $S_n$，则：

$$S_n = \frac{n(a_1 + a_n)}{2} = \frac{n[2a_1 + (n-1)d]}{2}$$

或写成：
$$S_n = na_1 + \frac{n(n-1)d}{2}$$

**两个公式的选择**：
- 若已知 $a_1$ 和 $a_n$：用 $S_n = \frac{n(a_1 + a_n)}{2}$
- 若已知 $a_1$ 和 $d$：用 $S_n = na_1 + \frac{n(n-1)d}{2}$

**重要结论**：
- $S_n$ 是 $n$ 的二次函数（当 $d \neq 0$ 时）
- 当 $d > 0$ 时，数列递增；当 $d < 0$ 时，数列递减；当 $d = 0$ 时，为常数列
- $S_n$ 与 $a_n$ 的关系：$a_n = S_n - S_{n-1}$ （$n \geq 2$）

### 2.5 例题

**例1**：已知等差数列 $\{a_n\}$ 中，$a_1 = 3$，$d = 2$，求 $a_{10}$ 和 $S_{10}$。

**解**：
$$a_{10} = a_1 + (10-1)d = 3 + 9 \times 2 = 21$$
$$S_{10} = \frac{10(3 + 21)}{2} = \frac{10 \times 24}{2} = 120$$

**例2**：已知等差数列 $\{a_n\}$ 中，$a_3 = 7$，$a_7 = 19$，求 $a_1$、$d$ 和 $S_{10}$。

**解**：
由通项公式：
$$a_3 = a_1 + 2d = 7$$
$$a_7 = a_1 + 6d = 19$$

两式相减：$4d = 12 \Rightarrow d = 3$

代入第一个式子：$a_1 + 6 = 7 \Rightarrow a_1 = 1$

因此：
$$S_{10} = 10 \times 1 + \frac{10 \times 9 \times 3}{2} = 10 + 135 = 145$$

## 三、等比数列 (Geometric Progression)

### 3.1 定义
如果一个数列从第二项起，每一项与前一项的比都等于同一个常数 $q$（$q \neq 0$），则称该数列为**等比数列**，$q$ 称为**公比**。

$$\frac{a_{n+1}}{a_n} = q \quad (n \in \mathbb{N}^*, \; a_n \neq 0)$$

### 3.2 通项公式

设等比数列首项为 $a_1$，公比为 $q$，则第 $n$ 项为：

$$a_n = a_1 \cdot q^{n-1}$$

**推导**：
- $a_1 = a_1$
- $a_2 = a_1q$
- $a_3 = a_1q^2$
- $\vdots$
- $a_n = a_1q^{n-1}$

### 3.3 性质

**性质1**：若 $m + n = p + q$，则 $a_m \cdot a_n = a_p \cdot a_q$

**推导**：
$$a_m \cdot a_n = a_1q^{m-1} \cdot a_1q^{n-1} = a_1^2 q^{m+n-2}$$
$$a_p \cdot a_q = a_1q^{p-1} \cdot a_1q^{q-1} = a_1^2 q^{p+q-2}$$

当 $m+n = p+q$ 时，两式相等。

**性质2**：三个数成等比数列 $\Leftrightarrow$ 中间一个是另两个的等比中项

即若 $a, b, c$ 成等比数列，则 $b^2 = ac$（且 $b \neq 0$）

**性质3**：等比数列中，若 $q = 1$，则所有项都相等；若 $q = -1$，则数列为 $a_1, -a_1, a_1, -a_1, \ldots$

### 3.4 前 $n$ 项和公式

设等比数列的前 $n$ 项和为 $S_n$，则：

**当 $q = 1$ 时**：
$$S_n = na_1$$

**当 $q \neq 1$ 时**：
$$S_n = \frac{a_1(1-q^n)}{1-q} = \frac{a_1(q^n-1)}{q-1}$$

**推导**（$q \neq 1$ 的情况）：
$$S_n = a_1 + a_1q + a_1q^2 + \cdots + a_1q^{n-1}$$
$$qS_n = a_1q + a_1q^2 + a_1q^3 + \cdots + a_1q^n$$

两式相减：
$$(1-q)S_n = a_1 - a_1q^n = a_1(1-q^n)$$
$$S_n = \frac{a_1(1-q^n)}{1-q}$$

### 3.5 例题

**例1**：已知等比数列 $\{a_n\}$ 中，$a_1 = 2$，$q = 3$，求 $a_5$ 和 $S_5$。

**解**：
$$a_5 = a_1 \cdot q^{5-1} = 2 \times 3^4 = 2 \times 81 = 162$$
$$S_5 = \frac{a_1(q^5-1)}{q-1} = \frac{2(3^5-1)}{3-1} = \frac{2 \times 242}{2} = 242$$

**例2**：已知等比数列 $\{a_n\}$ 中，$a_2 = 4$，$a_4 = 16$，求 $a_1$、$q$ 和 $S_6$。

**解**：
由通项公式：
$$a_2 = a_1q = 4$$
$$a_4 = a_1q^3 = 16$$

两式相除：$q^2 = 4 \Rightarrow q = \pm 2$

当 $q = 2$ 时：$a_1 = 2$
$$S_6 = \frac{2(2^6-1)}{2-1} = 2 \times 63 = 126$$

当 $q = -2$ 时：$a_1 = -2$
$$S_6 = \frac{-2[(-2)^6-1]}{-2-1} = \frac{-2 \times 63}{-3} = 42$$

## 四、等差数列与等比数列的比较

| 性质 | 等差数列 | 等比数列 |
|------|---------|---------|
| **定义** | $a_{n+1} - a_n = d$ | $\frac{a_{n+1}}{a_n} = q$ |
| **通项公式** | $a_n = a_1 + (n-1)d$ | $a_n = a_1 \cdot q^{n-1}$ |
| **前n项和** | $S_n = \frac{n(a_1+a_n)}{2}$ | $S_n = \frac{a_1(1-q^n)}{1-q}$ (当 $q \neq 1$) |
| **中项性质** | $2b = a + c$ | $b^2 = ac$ |
| **单调性** | 由 $d$ 符号决定 | 由 $a_1$ 和 $q$ 共同决定 |
| **增长速度** | 线性增长 | 指数增长 |

## 五、重要公式速查表

### 等差数列
- 通项：$a_n = a_1 + (n-1)d$
- 求和：$S_n = \frac{n(a_1+a_n)}{2}$ 或 $S_n = na_1 + \frac{n(n-1)d}{2}$
- 公差：$d = \frac{a_n - a_1}{n-1}$ 或 $d = \frac{a_n - a_m}{n-m}$
- 项数：若首尾两项已知，$n = \frac{a_n - a_1}{d} + 1$

### 等比数列
- 通项：$a_n = a_1 \cdot q^{n-1}$
- 求和（$q \neq 1$）：$S_n = \frac{a_1(1-q^n)}{1-q}$
- 求和（$q = 1$）：$S_n = na_1$
- 公比：$q = \frac{a_n}{a_1} \cdot \frac{1}{q^{n-2}} = \sqrt[n-1]{\frac{a_n}{a_1}}$ 或 $q = \frac{a_n}{a_m} \cdot \frac{1}{q^{n-m}}$

## 六、常见题型与解题技巧

### 6.1 求通项公式
1. **已知首项和公差（公比）**：直接代入通项公式
2. **已知两项**：列两个方程求出 $a_1$ 和 $d$（或 $q$）
3. **已知递推关系**：根据递推关系找规律

### 6.2 求和问题
1. **直接求和**：用求和公式
2. **错位相减法**：数列为等差乘等比数列
3. **分组求和法**：数列分成几个等差或等比数列

### 6.3 性质应用
1. 利用等差/等比中项简化计算
2. 利用项数关系：$m + n = p + q$ 的性质
3. 利用和与项的关系：$a_n = S_n - S_{n-1}$

## 七、综合应用题

**题1**：已知 $\{a_n\}$ 是等差数列，$\{b_n\}$ 是等比数列，且 $a_1 = b_1 = 1$，$a_2 + a_3 = 10$，$b_2 + b_3 = 12$，求两个数列。

**解**：
对于等差数列 $\{a_n\}$：
$$a_2 + a_3 = (a_1+d) + (a_1+2d) = 2a_1 + 3d = 2 + 3d = 10$$
$$\Rightarrow d = \frac{8}{3}$$

所以 $a_n = 1 + (n-1) \cdot \frac{8}{3} = \frac{8n-5}{3}$

对于等比数列 $\{b_n\}$：
$$b_2 + b_3 = q + q^2 = 12$$
$$q^2 + q - 12 = 0$$
$$(q+4)(q-3) = 0$$
$$q = 3 \text{ 或 } q = -4$$

所以 $b_n = 3^{n-1}$ 或 $b_n = (-4)^{n-1}$
